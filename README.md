## :toolbox: Getting Started

### :gear: Installation

Installation Steps
```bash
npm i
```


### :running: Run Locally

Clone the project

```bash
https://github.com/atheshwaran-10/Translingo
```
Go to API directory
```bash
cd api
```
Run the API server
```bash
node index.js
```
Go back to client directory
```bash
cd ../client
```
Run the React App
```bash
npm run dev
```


### :triangular_flag_on_post: Deployment

Build Command
```bash
npm run build
```
